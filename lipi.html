Shibulipi - A Writing System for Indic Languages

Shibulipi was invented when I was in Class VIII. It is a writing system for Indic languages, much in the same spirit as Nikhilipi or Bharathi Lipi. Based loosely on Bengali and Sanskrit writing systems, Shibulipi can be augmented and customized for any Indic language, including Tibetan and Thai.
